# LinkIt jQuery Plugin #
**LinkIt** is a simple jQuery plugin to attach links to elements

## Version ##
0.3.0

## Usage ##
    $('span').linkIt({
        href : 'https://test.com',
        text : 'click me',
        target : '_blank'
    });

## Vendors ##
jQuery - [https:/jquery.com](https:/jquery.com)
